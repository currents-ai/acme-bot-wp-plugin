# AcmeBot WordPress Plugin

AcmeBot is a WordPress plugin that allows you to automatically create posts using AI. This plugin is designed to help bloggers, journalists, and content creators generate high-quality articles quickly and efficiently.

## Features

- Create new WordPress posts with AI-generated content
- Supports custom post titles and categories
- Easy-to-use interface in the WordPress admin dashboard
- Option to review and edit generated content before publishing

## Installation

1. Download the `acmebot` plugin folder.
2. Upload the folder to your WordPress site's `wp-content/plugins/` directory.
3. Go to the WordPress admin dashboard.
4. Navigate to **Plugins** > **Installed Plugins**.
5. Activate **AcmeBot**.

## Usage

1. After activation, go to **AcmeBot** in the WordPress admin menu.
2. Enter your desired post title and optional keywords or prompts.
3. Click **Generate Post**.
4. Review the generated content.
5. Click **Publish** to add the post to your site.

## Requirements

- WordPress 5.0 or higher
- PHP 7.2 or higher
